Project Title
Users App

Project description:
The App fetches a list of users and displays them, on clicking on the user you can view the further details about user.  The app also allows to sort and search the users.

Technologies:
React js
Bootstrap
	
Installation & Run

Open folder cd-assignment in cmd.
You will need node and npm installed globally on your machine.

Installation:
npm install

To Start Server:
npm start

To Visit App:
localhost:3000
